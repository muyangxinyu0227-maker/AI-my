# Claude Code Skills 学习

## 什么是 Skills？

Skills 是一种将专业知识、工作流程和最佳实践打包成"可复用技能包"的功能。

## 为什么需要 Skills？

- **重复指令**：每次都要解释要求
- **知识无法沉淀**：团队经验无法共享
- **标准不统一**：结果可能完全不同
- **效率低下**：常见任务需要重复解释

## 安装 find-skills

```bash
npx skills add vercel-labs/skills@find-skills -g -y
```

## 常用 Skills

| Skill | 功能 |
|-------|------|
| find-skills | 技能搜索 |
| commit | 生成提交信息 |
| review | 代码审查 |
| test-generator | 测试生成 |

## 使用方法

直接告诉 Claude 你的需求：
