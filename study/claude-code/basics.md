# Claude Code 基础学习

## 什么是 Claude Code？

Claude Code 是 Anthropic 官方推出的 AI 命令行工具，它将大型语言模型的能力直接集成到终端中，让你可以用自然语言与 AI 协作完成编程任务。

## 快速安装

2.1.53 (Claude Code)

## 核心功能

- 读取和编辑本地文件
- 运行终端命令
- 使用 Bash 工具
- 理解项目上下文
