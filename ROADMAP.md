# 学习路线图 (ROADMAP)

> Claude Code 系统学习路线，4-8 周成为 Claude Code 高手

## 阶段一：入门基础（1周）

### Week 1: 环境搭建与基础操作

| 日期 | 主题 | 内容 | 练习 |
|------|------|------|------|
| Day 1 | 安装配置 | Node.js 安装、Claude Code 安装、账户配置 | 完成开发环境搭建 |
| Day 2 | 基础操作 | 对话交互、文件读取、代码编辑 | 修改一个现有文件 |
| Day 3 | 常用指令 | @语法、/命令、!Shell | 使用 @grep 搜索代码 |
| Day 4 | 项目探索 | @directory、代码理解 | 探索一个开源项目 |
| Day 5 | 实战练习 | 完成第一个任务 | 实现一个 Hello World |

**本周目标**：
- [ ] 安装并配置好 Claude Code
- [ ] 掌握基础操作命令
- [ ] 完成第一个开发任务

## 阶段二：核心技能（2周）

### Week 2: 工作流与 MCP

| 日期 | 主题 | 内容 | 练习 |
|------|------|------|------|
| Day 6 | 开发工作流 | 新功能开发流程、Bug修复流程 | 按流程开发一个功能 |
| Day 7 | 代码审查 | /review 命令、代码质量检查 | 审查自己代码 |
| Day 8 | Git 操作 | /commit、PR 创建 | 提交代码并创建 PR |
| Day 9 | MCP 入门 | 什么是 MCP、配置方法 | 配置 GitHub MCP |
| Day 10 | MCP 进阶 | 数据库、文件系统等 | 配置 SQLite MCP |
| Day 11 | 项目配置 | CLAUDE.md、settings.json | 配置项目环境 |
| Day 12 | 实战项目 | 综合练习 | 开发一个小工具 |

**本周目标**：
- [ ] 掌握标准开发工作流
- [ ] 配置并使用 MCP
- [ ] 完成一个完整项目

### Week 3: Skills 与 Superpowers

| 日期 | 主题 | 内容 | 练习 |
|------|------|------|------|
| Day 13 | Skills 基础 | 什么是 Skills、安装使用 | 安装 find-skills |
| Day 14 | Skills 进阶 | 常用技能、定制开发 | 开发一个自定义 Skill |
| Day 15 | Superpowers | 简介、安装 | 安装 Superpowers |
| Day 16 | TDD 开发 | 测试驱动开发、红绿重构 | 用 TDD 实现功能 |
| Day 17 | 需求澄清 | brainstorming 技能 | 练习需求分析 |
| Day 17 | 计划执行 | writing-plans 技能 | 制定项目计划 |
| Day 18 | 代码审查 | code-review 技能 | 完成代码审查 |

**本周目标**：
- [ ] 熟练使用 Skills
- [ ] 掌握 Superpowers
- [ ] 理解 TDD 开发流程

## 阶段三：高级进阶（3周）

### Week 4: Agent Teams 与规范

| 日期 | 主题 | 内容 | 练习 |
|------|------|------|------|
| Day 19 | Agent Teams | 概念、使用场景 | 创建团队任务 |
| Day 20 | 并行开发 | 多代理协作 | 并行开发前后端 |
| Day 21 | 规范编程 | 代码规范、文档 | 编写项目规范 |
| Day 22 | 测试策略 | 单元测试、集成测试 | 编写完整测试 |
| Day 23 | 调试技巧 | 断点、日志 | 调试复杂 Bug |
| Day 24 | 重构实践 | 安全重构 | 重构旧代码 |
| Day 25 | 实战项目 | 综合练习 | 开发完整项目 |

### Week 5: 专题深入

| 日期 | 主题 | 内容 | 练习 |
