# Claude Agent Teams 学习

## 什么是 Agent Teams？

Agent Teams 让多个独立的 AI 实例可以像真正的开发团队一样协同工作。

## 单 AI 模式的局限

- **串行处理**：一次只能做一件事
- **上下文拥挤**：对话过长会遗忘重要信息
- **单一视角**：缺乏多角度讨论
- **效率瓶颈**：无法并行加速

## Agent Teams 的优势

- **真正并行**：多个 AI 同时处理不同任务
- **独立上下文**：每个成员有 200K token 上下文
- **团队协作**：成员之间可以直接交流
- **效率提升**：大型项目效率提升约 50%

## Agent Teams vs Subagent

| 对比 | Subagent | Agent Teams |
|------|----------|-------------|
| 拓扑结构 | 星型（向主代理汇报） | 网状（互相通信） |
| 通信方式 | 单向 | 双向 |
| 上下文 | 共享 | 完全独立 |
| 并行能力 | 有限 | 真正并行 |

## 使用场景

- 前端 + 后端并行开发
- 多个模块同时重构
- 大型项目的分工协作
