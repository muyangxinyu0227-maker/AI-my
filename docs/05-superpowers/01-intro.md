# Superpowers 简介

## 什么是 Superpowers？

Superpowers 是 Jesse Vincent 开发的技能框架，让 AI 写出工程级代码。

## 与直接使用的区别

| 维度 | 直接使用 | Superpowers |
|------|---------|-------------|
| 需求澄清 | 直接开始 | 苏格拉底式提问 |
| 开发流程 | 自由发挥 | 强制 TDD |
| 任务管理 | 一次性完成 | 分解小任务 |

## 核心技能

| 技能 | 功能 |
|------|------|
| brainstorming | 需求澄清 |
| test-driven-development | TDD 开发 |
| writing-plans | 制定计划 |
| systematic-debugging | 系统调试 |

## 触发关键词

| 技能 | 触发词 |
|------|--------|
| TDD | "TDD"、"测试驱动" |
| brainstorming | 需求模糊时自动 |
| debugging | "调试"、"bug" |
