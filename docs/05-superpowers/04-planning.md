# 计划执行

## writing-plans 技能

将大任务分解为小步骤。

## 使用方法

```
用 writing-plans 规划一个待办事项 API
```

## 输出示例

```markdown
# 实现计划

## 任务 1：设计数据库（5分钟）
- 创建 todos 表
- 定义字段

## 任务 2：创建路由（10分钟）
- POST /todos
- GET /todos

## 任务 3：添加验证（10分钟）
- 标题不能为空

## 任务 4：编写测试（15分钟）
- 为每个端点测试

## 任务 5：启动验证（5分钟）
- 运行测试
- 手动测试
```

## 任务特点

- 每个任务 2-5 分钟
- 任务独立可验证
- 按顺序执行

## executing-plans

执行计划并在每个检查点暂停：

```
执行上面的计划，每完成一个任务暂停
```
