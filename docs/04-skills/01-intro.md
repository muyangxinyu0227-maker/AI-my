# Skills 简介

## 什么是 Skills？

Skills 是可复用的技能包，让 Claude 知道如何处理特定任务。

## 安装 find-skills

```bash
npx skills add vercel-labs/skills@find-skills -g -y
```

## 使用 find-skills

```
我需要做 React 性能优化，找找有什么技能
```

## 常用 Skills

| Skill | 功能 |
|-------|------|
| find-skills | 搜索技能 |
| commit | 生成提交信息 |
| review | 代码审查 |
| test-generator | 生成测试 |

## Skills.sh

访问 https://skills.sh 浏览可用技能。
