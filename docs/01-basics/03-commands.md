# 基础命令

## 文件操作

### 读取文件

```text
@filename
```

或直接描述：

```
读取 src/index.js 文件
```

### 编辑文件

```
把 src/index.js 中的 foo 函数改名为 bar
```

### 创建文件

```
创建一个新的组件文件 src/Button.tsx
```

## 代码搜索

### 搜索内容

```
@grep "functionName" --glob "*.ts"
```

### 查找文件

```
找一下项目中的配置文件
```

## 目录操作

### 查看目录

```
@directory src/
```

### 列出文件

```
列出当前目录的文件
```

## 命令执行

### 运行 Shell

```text
!npm test
```

或描述：

```
运行项目的测试
```

## 常用快捷命令

| 命令 | 功能 |
|------|------|
| @文件路径 | 读取/引用文件 |
| @grep "关键词" | 搜索代码 |
| @directory 路径 | 查看目录 |
| !命令 | 执行 Shell |
| /help | 查看帮助 |
| /commit | 提交代码 |
| /review | 代码审查 |
| /plan | 制定计划 |

## 实战练习

1. 使用 `@directory` 查看项目结构
2. 使用 `@grep` 搜索一个函数
3. 使用 `!` 运行一个命令
4. 使用 `/help` 查看所有命令
