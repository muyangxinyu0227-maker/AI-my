# 基础概念

## 什么是 Claude Code？

Claude Code 是 Anthropic 推出的 AI 编程助手，它：
- 集成在终端中
- 理解整个项目上下文
- 可以执行复杂开发任务
- 支持代码生成、重构、调试

## 核心组件

### 1. 对话界面

Claude Code 提供交互式对话界面，你可以：
- 用自然语言描述需求
- 询问代码问题
- 请求代码审查

### 2. 工具系统

Claude Code 可以使用的工具：

| 工具 | 功能 |
|------|------|
| Read | 读取文件内容 |
| Edit | 编辑文件 |
| Write | 创建新文件 |
| Bash | 执行终端命令 |
| Grep | 搜索代码 |
| Glob | 查找文件 |

### 3. @语法

- `@文件名` - 引用文件
- `@directory 路径` - 查看目录
- `@grep "关键词"` - 搜索代码

### 4. /命令

- `/help` - 查看帮助
- `/commit` - 提交代码
- `/review` - 代码审查
- `/plan` - 制定计划

## 工作原理

```
用户输入 → Claude Code → 分析理解 → 调用工具 → 返回结果
```

## 上下文管理

Claude Code 会记住：
- 当前项目的结构
- 对话历史
- 之前修改的内容
