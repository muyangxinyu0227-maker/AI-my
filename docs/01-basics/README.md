## 快速开始

### 安装 Claude Code

```bash
# 检查 Node.js 版本
node --version  # 需要 v18+

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version
```

### 首次启动

```bash
# 进入项目目录
cd your-project

# 启动 Claude Code
claude
```

首次启动会引导你完成账户配置。
