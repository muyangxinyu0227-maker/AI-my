## 概述

## MCP 的作用

### 没有 MCP

- ✅ 读取本地文件
- ✅ 编辑代码
- ✅ 运行命令
- ❌ 查看 GitHub Issues
- ❌ 访问数据库

### 有了 MCP

- ✅ 所有原有功能
- ✅ GitHub 集成
- ✅ 数据库查询
- ✅ 浏览器自动化
